What is PopForge?

PopForge is an ISO to PBP converter GUI built for retro gaming users who want a cleaner way to prepare PS1 games for PSP and PS Vita Adrenaline. It wraps the popstationr conversion engine and aims to provide a simple interface for converting PSX disc images into the EBOOT.PBP format.

Many older PSP PBP converter tools are Windows-only, require outdated runtimes, or rely on command-line workflows. PopForge is being built as a modern cross-platform alternative for macOS Apple Silicon, Windows, and Linux.

Why convert PS1 ISO files to PBP?

PSP and PS Vita Adrenaline use EBOOT.PBP packages for many PlayStation 1 games. Converting an ISO, BIN/CUE, or IMG disc image to PBP can make it easier to organize PS1 games for PSP-style libraries while keeping each game in a single package.

PopForge is designed for users searching for a simple ISO to PBP converter, PSX to PBP converter, or PSP EBOOT.PBP converter with a desktop interface.

PopForge vs older PBP converter tools

Older tools like ISO-PBP Converter, PSX2PSP, and some PSXPackager workflows can be useful, but many are dated, Windows-focused, or require extra runtime setup. PopForge aims to provide a lightweight modern GUI for common PS1 to PSP conversion tasks.

PopForge is built on top of popstationr, a lightweight command-line tool for creating PSP-compatible PS1 EBOOT files, and is inspired by PSXPackager.

FAQ

How do I convert ISO to PBP for PSP?

PopForge is being built to let you choose an ISO, BIN/CUE, or IMG file, add it to a batch queue, and convert it into a PSP-compatible EBOOT.PBP file.

What is an EBOOT.PBP file?

EBOOT.PBP is a package format used by PSP software. For PS1 games, it can contain the converted game image and metadata used by PSP and PS Vita Adrenaline.

Can PopForge convert BIN/CUE to PBP?

BIN/CUE to PBP support is part of the planned PopForge workflow for PlayStation 1 and PSX disc images.

Does PopForge work on macOS?

PopForge is planned for macOS Apple Silicon, Windows, and Linux. Intel Mac builds are not currently planned.

Is PopForge a PSXPackager alternative?

PopForge is inspired by PSXPackager, but the goal is a minimal cross-platform GUI focused on ISO to PBP, PSX to PBP, and PSP EBOOT.PBP workflows.

Is it PBP or BPB?

The correct file extension is PBP. Some users mistype it as BPB, but PSP EBOOT files use the .PBP extension.
